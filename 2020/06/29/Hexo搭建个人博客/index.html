使用Github Pages + Hexo搭建个人博客

Posted on 2020-06-29 Edited on 2021-07-17 In Github Views:
网上有很多使用 Github Pages 搭建个人博客的教程，大部分是配合 Hexo 和 Jekyll 来搭建。这篇文章是我找到的讲得非常清楚的一篇博客，从在 Github Pages 建站到 Hexo 美化，再到配置域名，对小白非常友好。本个人博客即是参照该文章搭建的（当然还找了一些其他的 Hexo 教程进行了美化）。

一、建站

网上有很多使用 Github Pages 搭建个人博客的教程，大部分是配合 Hexo 和 Jekyll 来搭建。这篇文章是我找到的讲得非常清楚的一篇博客，从在 Github Pages 建站到 Hexo 美化，再到配置域名，对小白非常友好。

本个人博客即是参照该文章搭建的（当然还找了一些其他的 Hexo 教程进行了美化）。 欢迎大家访问我的博客：https://www.charloe.top/

二、注意事项

在这里，我想讲一讲建站过程中的一些注意事项和我遇到的问题。

1.安装完 Hexo 后， 在 Git 中输入相关命令时比如：hexo g -d，我的命令需要改为 npx hexo g -d 才行，这可能是因为我没有设置环境变量。

2.原博客中给出了两种解析域名的方法。我是在腾讯云上购买了 charloe.top 域名，当我使用方法一时，我发现我还是无法通过 charloe.top 或 www.charloe.top 来进行访问，甚至当我使用 username.github.io 时都无法访问。然后我使用了腾讯云中的域名检测功能，提示未找到 A 类型记录。我想到原博客中方法一使用的是 CNAME 类型，而方法二使用的是 A 类型。最后，我使用了方法二可以成功访问。
